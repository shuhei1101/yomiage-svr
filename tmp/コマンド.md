
## venv作成
```sh
python -m venv venv
```

## pip インストール
```sh
pip install -r requirements.txt
```

## requirements.txt 更新
```bash
pip freeze > requirements.txt
```

## venv有効化
```sh
source venv/bin/activate
```


## ポートからタスクキル
```sh
port=8767
lsof -ti:$port | xargs kill -9
```
## 読み上げkill
```sh
port=50021
lsof -ti:$port | xargs kill -9
```

## 現在の動作中のタスクとポートを確認
```sh
lsof -iTCP -sTCP:LISTEN -n -P
```


## pythonkill
```sh
pkill -f python
```


## サーバ起動
```sh
./start_server.sh
```
